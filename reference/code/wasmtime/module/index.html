Module

A Module represents WASM code that has been compiled by an Engine into executable machine code. The Module class is a wrapper around the wasmtime::Module struct. A Module declares a set of imports (things it requires) and exports (things it provides).

The Engine which is used controls exactly how the WASM is compiled into a Module. For example if the Engine has fuel usage enabled the Module will be compiled with support for fuel usage. It is possible to have multiple modules which represent the same WASM code compiled with different engines. An Engine in Wasmbox is usually configured with an EngineConfig.

Linker

A Linker provides a set of imports for instantiating a Module. The Linker class is a wrapper around the wasmtime::Linker struct.

A Module contains a set of imports, such as functions the WASM code can call, which must be supplied when an Instance is created from the Module. A Linker provides a convenient way to do this.
